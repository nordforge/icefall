# IF-234: Ghost Mode UI indicators

**Phase:** 29 — Frontend UI
**Priority:** Medium
**Estimate:** S

## Description

Surface Ghost Mode status (IF-183) in the dashboard. Hibernating apps show a distinct state.

## Acceptance Criteria

- [ ] App card: "Hibernating" status with moon icon (distinct from stopped)
- [ ] App detail overview: Ghost Mode status section
- [ ] "Wake" button when hibernating
- [ ] Settings tab: Ghost Mode toggle + idle timeout config
- [ ] Dashboard: "Ghost Mode savings" metric (estimated RAM saved)
- [ ] a11y: status announced to screen readers

## Dependencies

- IF-183 (Ghost Mode backend)

# IF-049: Logo, social card, and brand assets

**Phase:** 12 — Landing & Docs
**Priority:** Medium
**Estimate:** S

## Description

Create the Icefall visual identity: logo, social card for OG tags, and basic brand assets.

## Acceptance Criteria

- [ ] Logo: simple, sharp mark that works at 16px (favicon) and 200px+ (landing page)
  - Glacial/ice theme, not literal (no snowflake clipart)
  - Works on both light and dark backgrounds
  - SVG format, with PNG exports at standard sizes
- [ ] Favicon: `.ico` + `.svg` + `apple-touch-icon.png`
- [ ] Social card (OG image): 1200x630, used for link previews on Twitter/LinkedIn/Slack
  - Project name, tagline, subtle brand visual
  - Light and dark variants
- [ ] GitHub social preview: 1280x640
- [ ] Color usage guidelines (which palette colors for brand contexts)
- [ ] Assets stored in `assets/brand/` in the repo

## Dependencies

- None (can be done in parallel with anything)

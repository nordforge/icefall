## Authentication

All API requests require authentication via:
- **Bearer token**: `Authorization: Bearer icefall_your_token`
- **Session cookie**: `Cookie: session_id=your_session`

Create API tokens at **Settings > API Tokens**.

## Response format

Success: `{ "data": ... }`
Error: `{ "error": { "code": "...", "message": "..." } }`

## Apps

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/v1/apps` | List all apps |
| POST | `/api/v1/apps` | Create an app |
| GET | `/api/v1/apps/{id}` | Get app details |
| PUT | `/api/v1/apps/{id}` | Update app settings |
| DELETE | `/api/v1/apps/{id}` | Delete an app |
| POST | `/api/v1/apps/{id}/start` | Start app |
| POST | `/api/v1/apps/{id}/stop` | Stop app |
| POST | `/api/v1/apps/{id}/restart` | Restart app |
| GET | `/api/v1/apps/{id}/drift` | Check config drift |
| GET | `/api/v1/apps/{id}/export` | Export as bundle |
| POST | `/api/v1/apps/{id}/clone` | Clone app |
